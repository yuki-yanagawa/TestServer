*********************************************************************************
操作
**********************************************************************************\
コマンドプロンプトより
start(packageExecute).bat、もしくは、start(classFileExecute).batを実行すると
=> HTTP Serverが起動します。

初期port番号は「12345」、ipAdressは「localhost(127.0.0.1)」となっております。

ブラウザを起動し、http://localhost:12345リクエストする形になります。

http://localhost:12345のリクエストは、index.htmlをレスポンスする形になります。

html配下に
htmlファイルを追加可能です

css配下に
cssファイルを追加可能です

javascript配下に
jsファイルを追加可能です

※htmlファイル、cssファイル、jsファイルをリクエストの際は
  フォルダ名を付与せず、ファイル名のみリクエストするようにお願いします。




*********************************************************************************
設定ファイル
**********************************************************************************\

conf配下にある
common.propertiesで設定変更可能になります。
↓↓↓↓
#### Http Server Setting
ipAddrUse=false
portNo=12345
recvBufferSize=1024


ipAddrUse=trueの場合は、設定されたIPアドレスで接続可能になります
portNoは設定したport番号で接続可能になります
recvBufferSizeは一度のパケットで受信可能サイズになります
